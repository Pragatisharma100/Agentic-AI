# Class 1 — File list and summaries

This folder contains the actual Class 1 code files.

## Files and short summaries

- [`currency.py`](./currency.py) — currency conversion and related utility examples.
- [`functions or tools.py`](./functions%20or%20tools.py) — helper functions and tools used in examples.
- [`Python_foundamentals.py`](./Python_foundamentals.py) — basic Python concepts and examples.
- [`tools.py`](./tools.py) — additional tool and utility examples.
- [`first_project/`](./first_project/) — a small project folder with its own `main.py` and `pyproject.toml`.

## How to open files

Click any linked file name above in VS Code to open it.

## How to run files

Use Python or `uv run` on the file name, for example:

```bash
uv run "Python_foundamentals.py"
uv run "currency.py"
```

## Notes

- `first_project/` is a separate example project with its own configuration.
- Keep file names and paths exact, especially when spaces are included.
