# Agentic AI

A modern, direct path to mastering Agentic AI and GenAI with production-ready multi-agent systems on AWS, GCP, Azure, and VPS.

---

## Overview

Agentic AI 3.0 is a hands-on learning path designed to take you from foundational concepts all the way to deploying Agentic AI systems in the cloud. This project captures a full course structure, roadmap, and practical examples for learning one step at a time.

### What you will learn

- Agentic AI fundamentals and GenAI workflows
- Agent frameworks and ecosystem tools
- Memory, knowledge, and agent-to-agent protocols
- Deployment, AgentOps, and observability
- Cloud-native production builds on AWS, GCP, Azure, and VPS

### Who this is for

- Learners moving from GenAI to Agentic AI
- Python developers seeking production-ready AI systems
- Engineers targeting multi-agent applications and cloud deployment

## Roadmap

The course is organized into five phases, each building on the prior phase with practical modules, tools, and projects.

### Phase 0 — Foundations (Weeks 1–3)
- 4 modules
- Python essentials, async patterns, GenAI fundamentals
- Prompt engineering, structured prompts, Pydantic
- Output: foundation for agent design and structured workflows

### Phase 1 — Core Frameworks & Protocols (Weeks 4–9)
- 9 modules + 2 projects
- LangChain, LCEL, LangGraph, multi-agent architecture
- MCP, agent protocols, Agentic RAG, context engineering
- Output: production-ready agent workflows

### Phase 2 — Claude Ecosystem & Coding Agents (Weeks 10–13)
- 6 modules + 1 project
- Claude Code, OpenClaw, NemoClaw, Hermes Agent
- Codex CLI, opencode, Antigravity IDE
- Persistent memory with mem0 and self-improving agents

### Phase 3 — Extended Frameworks (Weeks 14–17)
- 7 modules + 3 mini-projects
- OpenAI Agents SDK, CrewAI, Google ADK, AWS Strands
- LlamaIndex workflows, n8n, LangFlow
- Output: cross-framework agent solutions

### Phase 4 — Production Cloud Projects (Weeks 18–22)
- 1 module + 3 projects
- AgentOps CI/CD, observability, evaluation, cost control
- Deploy to Bedrock, Vertex AI, AI Foundry
- Portfolio-grade cloud projects with Docker, FastAPI, GitHub Actions, BentoML
